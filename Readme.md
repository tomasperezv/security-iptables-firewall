About
=============
Basic iptables rules for a custom firewall.

Usage
------------
It needs iptables: http://www.netfilter.org/projects/iptables/

Turn on the rules

```shell
$ on_rules.sh
```

Turn off the rules

```shell
$ off_rules.sh
```

Disconnect all of the network interfaces

```shell
$ sh disconnect.sh
```
